scroll-model.js

A library made to make award winning 3d scroll animation without the need of writing Javascript or CSS



<--import the component-->

<script type="module" src="https://unpkg.com/@google/model-viewer/dist/model-viewer.min.js"></script>
<script type="module" src="https://scroll-model.onrender.com/scroll-model.js"></script>

<--use it like any tag-->

<scroll-model 
    src="fallout_car.glb"
    camera-controls
    rotation-frames="0:0,0,0;
                    1:360,360,360">
</scroll-model>

Documentation

How to use

first import the libraries


<script type="module" src="https://unpkg.com/@google/model-viewer/dist/model-viewer.min.js"></script>
<script type="module" src="scroll-model.js"></script>

then use it just like model-viewer by google but just add "-frames" after the attributes to make it run using keyframes


<scroll-model 
src="fallout_car.glb"
camera-controls
rotation-frames="0:0,0,0;
                1:0,0,360">
</scroll-model>

here 0 means 0% page scrolled and 1 means 100% page scrolled

Attributes

  • src: the source of the 3d model
  • camera-controls: to enable camera controls
  • rotation-frames: keyframes for rotation in the format "keyframe:value,value,value;"
  • position-frames: keyframes for position in the format "keyframe:value,value,value;"
  • scale-frames: keyframes for scale in the format "keyframe:value,value,value;"
